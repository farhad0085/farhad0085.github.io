Personal Trainer Planner - Flask App

This is a Web Application built with Flask and Python. The basic idea was a trainer used a gym’s facility to train his/her customer, and the gym owner charged the trainer for using their facility. Features: 1. 3 kinds of users a. Admin b. Gym Owner c. Trainer 2. Different users have different login panels and different dashboards. 3. Only Gym owners can create an account and can add a trainer to their gym. 4. After adding the trainer the trainer will get an email with the credentials to login. 5. Once the trainer account is created, other gyms can add this trainer to their gym if they wish. 6. Gym owners can add different gyms, like if a gym company has 3 branches of gym then they can add them individually. 7. Gym owner can set their gym’s opening and closing time and also ‘off day’ 8. Gym owners can add booking on behalf of any trainer. 9. Gym owners can remove any trainer from their gym. 10. Facility has maximum capacity, e.g 20 so at a time more than 20 people cannot train. If already 20 people are set, then new booking can’t be created for that time. 11. Trainers can book their schedule. 12. Have a calendar feature where users can check which day they have which schedule. 13. Users can update their profile anytime. 14. Admin can view and delete bookings, trainers and gym owners. Please watch the attached video to view full application's review. In the backend the application uses Flask and in frontend Bootstrap and native JS.
